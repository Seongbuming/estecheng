문의하기

내진성능평가 절차

예비평가

  1. 정형건물만이 예비평가 대상 (비정형 건물은 상세평가 실시)
  2. 시설안전공단에서 제시한 수식 적용
  3. 건물의 층별 성능 (Capacity) 산정
  4. 건물의 층별 요구치 (Demand) 산정
  5. 성능수준은 DCR= Demand/Capacity의 비에 따라 판정
성능수준 DCR의 범위 비고
거주가능 (IO) DCR ≤ 0.5
인명안전 (LS) 0.5 ＜ DCR ≤ 0.75 관공서
붕괴방지 (CP) 0.75 ＜ DCR ≤ 1.0
붕괴 (C) 1.0 ＜ DCR

상세평가 1단계

  1. 해석모델을 이용한 선형 (탄성)해석 실시
  2. 부재별 성능수준과 층별 성능수준을 바탕으로 전체 구조물 성능수준 판정
  3. 성능수준의 판정
성능수준 판정기준
거주가능 (IO) 부재별 성능수준이 즉시거주인 부재가 연직하중 분담률 80% 이상
인명안전 (LS) 부재별 성능수준이 인명안전, 즉시거주인 부재가 부담하는 연직하중 분담률 80% 이상
붕괴방지 (CP) 부재별 성능수준이 붕괴방지, 인명안전, 즉시거주인 부재가 부담하는 연직하중 분담률 80% 이상
붕괴 (C) 부재별 성능수준이 붕괴위험인 부재가 부담하는 연직하중 분담률 80% 이상

상세평가 2단계

  1. 해석모델을 이용한 비선형 해석 실시
  2. 성능점 (performance point)산정
  3. 성능점에서의 부재별 성능수준과 층별 성능수준을 바탕으로 전체 구조물 성능수준 판정
  4. 성능수준의 판정은 1단계 성능평가방법과 동일

내진성능 향상

  1. 최소 보강개소를 적용하여 상세평가 2단계 실시
  2. 목표성능 (인명안전) 만족 시 종료
  3. 목표성능 미 만족시 보강개소를 증가시켜 상세평가 2단계 실시
  4. 목표성능을 만족할 때까지 해석 반복 수행
